# BTT Forum Toolkit  
 BTT Forum için alıntı, tema gibi araçları barındıran basit bir toolkit eklentisi. 
 
Adı BTT Toolkit Firefox magazaya kaydini gönderdim şuan .xpi olarak dağıtıyorum.
Firefox'a Kur (.xpi):
https://sxi.is-a.dev/btt/btt-v2.2.xpi

Site (updates.json): https://sxi.is-a.dev/btt
Github (source code): https://github.com/sxinar/btt

## Özellikler:
- Sohbette Alıntıla Düğmesi (Artık fish sinirlenmeyecek :D)
- Konulardaki yanıtla düğmesinin alıntılama özelliği
- Tema seçenekleri
- Updates özelliği ile otomatik update sistemi
